## 整体布局结构

Squirrel 运维平台采用经典的侧边栏 + 主内容区布局模式，整体结构如下：

```
main-layout (根容器)
├── sidebar (侧边栏 - 固定宽度 200px)
│   ├── logo (Logo 区域)
│   └── nav (导航菜单)
└── main-content (主内容区 - 自适应宽度)
    ├── header (顶部栏 - 固定高度)
    │   ├── header-left (页面标题)
    │   └── header-right (用户信息 + 退出按钮)
    └── content (页面内容 - 可滚动)
```

## 侧边栏组件

### Logo 区域

**功能说明**：
- 展示 Squirrel 品牌 Logo 和名称
- 位于侧边栏顶部
- 固定高度，不随导航滚动

**结构组成**：
```
logo
├── logo-icon (Logo 图标)
└── logo-text (Logo 文字)
```

**设计要点**：
- Logo 图标使用 `lucide:squirrel` 图标
- 图标与文字左对齐，保持 8px 间距
- 整体区域高度约 44px

### 导航菜单

**功能说明**：
- 提供主要功能模块的导航入口
- 支持路由高亮和悬停效果
- 垂直排列，支持滚动

**结构组成**：
```
nav
└── nav-item (导航项 - 多个)
    ├── nav-icon (导航图标)
    └── nav-text (导航文字)
```

**导航项配置**：
每个导航项包含以下属性：
- `path`: 路由路径
- `label`: 显示文字
- `icon`: 图标名称（使用 lucide 图标集）

**状态说明**：
- **默认状态**: 半透明白色文字
- **悬停状态**: 浅蓝色背景，文字变亮，右移 2px
- **激活状态**: 渐变蓝色背景，白色文字，带阴影

## 顶部栏组件

### 页面标题区域

**功能说明**：
- 显示当前页面的标题
- 根据路由自动更新
- 位于顶部栏左侧

**设计要点**：
- 标题字号 18px，字重 600
- 颜色使用深蓝色 `#1e3a5f`
- 与左侧保持 10px 间距

### 用户信息区域

**功能说明**：
- 显示当前登录用户信息
- 位于顶部栏右侧
- 包含用户图标和用户名

**结构组成**：
```
user-info
├── user-icon (用户图标)
└── user-name (用户名)
```

**设计要点**：
- 背景色 `#f5f7fa`，圆角 12px
- 图标与文字间距 6px
- 整体内边距 6px 12px

### 退出按钮

**功能说明**：
- 提供用户退出登录功能
- 位于用户信息右侧
- 支持悬停效果

**设计要点**：
- 正方形按钮，尺寸 32px × 32px
- 默认灰色背景，悬停时变红色
- 悬停时旋转 -5deg 增强交互感

## 主内容区域

### 内容容器

**功能说明**：
- 承载具体页面的内容
- 支持垂直滚动
- 自适应剩余空间

**设计要点**：
- 内边距 20px
- 背景色 `#f5f7fa`
- 滚动条样式需与整体风格一致

## 响应式考虑

当前设计主要针对桌面端，如需支持移动端，建议：

1. **侧边栏**：在小屏幕下可折叠为图标模式或抽屉式
2. **顶部栏**：保持固定，但可简化用户信息显示
3. **内容区**：调整内边距以适应小屏幕

## 可访问性要点

- 所有交互元素需支持键盘导航
- 图标需配置 `aria-label` 属性
- 颜色对比度需符合 WCAG AA 标准
- 焦点状态需有明显视觉提示
