# Squirrel 登录页面设计

## 触发条件（When）

- 用户请求创建登录页面
- 用户需要参考登录页面设计规范
- 用户需要实现登录功能
- 用户询问登录页面的样式或布局

## 执行步骤（How）

1. 确认用户需求：是新建登录页面还是修改现有页面
2. 参考 [设计规范](design-spec.md) 确定配色、布局、组件样式
3. 参考 [代码实现](code-implementation.md) 获取完整代码模板
4. 根据项目实际情况调整代码

## 输出结果（What）

- 符合 Squirrel 设计规范的登录页面
- 包含品牌展示区、登录表单区
- 响应式布局，支持移动端
