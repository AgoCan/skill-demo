# 字体规范

## 字号层级

| 用途 | 字号 | 行高 | 字重 |
|------|------|------|------|
| 页面标题 | 18px | - | 600 |
| 卡片标题 | 14px | - | 600 |
| 正文文本 | 13px | - | 500 |
| 小文本 | 12px | - | 500 |
| 极小文本 | 11px | - | 500 |

## 文本颜色

```css
/* 主标题 */
color: #1e3a5f;

/* 正文 */
color: #64748b;

/* 浅色文本 */
color: #94a3b8;

/* 白色文本 */
color: #ffffff;
```
