# 紧凑风格页面设计规范

## 触发条件（When）

当用户需要处理页面样式相关问题时调用，包括：
- 创建新页面
- 调整现有页面样式
- 添加新组件样式
- 保持页面风格一致性

## 执行步骤（How）

### 1. 遵循设计原则

| 原则 | 说明 |
|------|------|
| 紧凑 | 减小字体、间距、圆角，提高信息密度 |
| 一致 | 所有页面使用统一的样式标准 |
| 轻盈 | 使用浅色背景和柔和阴影 |
| 现代 | 渐变、圆角、微交互 |

### 2. 应用样式规范

| 规范 | 说明 |
|------|------|
| 字体 | 页面标题 18px、正文 13px、小文本 12px |
| 间距 | 页面 16px、卡片 12px、表单 6px |
| 圆角 | 卡片 8px、按钮 6px、徽章 12px |
| 配色 | 主色 #4fc3f7、成功 #16a34a、危险 #dc2626 |

### 3. 使用组件样式

- 卡片：白色背景、8px圆角、柔和阴影
- 按钮：渐变背景、6px圆角、悬停效果
- 表单：浅色背景、聚焦高亮
- 徽章：圆角胶囊、状态颜色

## 输出结果（What）

统一风格的页面样式，包含：
- 紧凑的字体和间距
- 一致的圆角和阴影
- 协调的配色方案
- 流畅的过渡动画

## 样式速查

### 字号

| 用途 | 字号 |
|------|------|
| 页面标题 | 18px |
| 卡片标题 | 14px |
| 正文文本 | 13px |
| 小文本 | 12px |

### 间距

| 用途 | 值 |
|------|-----|
| 页面间距 | 16px |
| 卡片间距 | 12px |

### 圆角

| 元素 | 圆角 |
|------|------|

| 按钮 | 6px |
| 徽章 | 12px |
